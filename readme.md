# My Dashboard

Веб-приложение для управления дашбордом. Проект интегрирует Notion, GitHub и Cursor AI в единый рабочий процесс.

## Цели проекта

- Создать работающий прототип веб-приложения
- Интегрировать Notion, GitHub и Cursor AI в рабочий процесс
- Научиться эффективно использовать AI для разработки

## Ближайшие задачи

1. Настроить окружение разработки
2. Создать базовую структуру проекта в GitHub
3. Написать документацию в Notion и синхронизировать её с репозиторием

## Структура проекта

```
My_Dashboard/
├── src/      # Исходный код приложения
├── public/   # Статические файлы (изображения, шрифты и т.д.)

```
